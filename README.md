# LuLu's YouTube 下載寶

輕鬆下載 YouTube 影片和音訊的工具 💕

## 📋 系統需求

- Windows 系統
- Python 3.7 或以上版本
- Flask 套件

## 🚀 使用方式

### 第一次使用（安裝套件）

1. 開啟命令提示字元（cmd）或 PowerShell
2. 切換到此目錄
3. 執行安裝指令：
   ```
   pip install flask
   ```

### 啟動程式

1. 雙擊 `啟動.bat` 檔案
2. 等待伺服器啟動（會顯示「正在啟動伺服器...」）
3. 開啟瀏覽器，輸入網址：`http://localhost:5000`
4. 開始使用！

## 📖 功能說明

### 基本功能
- ✅ 下載 YouTube 影片（支援多種畫質）
- ✅ 下載 YouTube 音訊（僅音樂）
- ✅ 任務排隊系統（可同時加入多個下載）
- ✅ 即時進度顯示

### 進階功能
- 🔄 失敗任務重試（點擊 🔄 按鈕）
- ✕ 取消下載中任務（點擊 ✕ 按鈕）
- 🗑️ 清除已完成/失敗任務
- ⚙️ 設定錯誤重試上限（防止無限重試）

## 💡 使用小技巧

1. **選擇下載類型**
   - 🎥 影片：下載完整影片（含聲音）
   - 🎵 音訊：只下載音樂（mp3 格式）

2. **品質設定**
   - 影片：最佳品質、1080p、720p、480p
   - 音訊：最佳品質、320kbps、256kbps、192kbps、128kbps

3. **進階設定**
   - 點擊「⚙️ 進階設定」可設定錯誤重試上限
   - 預設為 0（不限制），可依需求調整

## ❓ 常見問題

**Q: 下載失敗怎麼辦？**
A: 點擊失敗任務右側的 🔄 按鈕重試，或檢查網路連線和影片連結。

**Q: 下載到哪裡了？**
A: 所有檔案都在 `downloads` 資料夾內。

**Q: 可以同時下載多個嗎？**
A: 可以！程式會依序處理，不用等待前一個完成。

**Q: 如何停止程式？**
A: 在命令提示字元視窗按 `Ctrl+C`，或直接關閉視窗。
